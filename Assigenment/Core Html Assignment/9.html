02 Week - Core Html Assignment

  1. <strong>
    • Use to emphasize text, making it bold.
    • Best for indicating strong importance or relevance.
    • Avoid using for purely visual styling; prefer CSS for styling text.
  2. <em>
    • Use to emphasize text, making it italicized.
    • Best for indicating stress or importance within text.
    • Avoid using for purely visual styling; prefer CSS for styling text.
  3. <u>
    • Use to underline text.
    • Best for indicating links or proper nouns in some cases.
    • Avoid using for non-link text, as underlining is traditionally associated with links.
  4. <del>
    • Use to mark text that has been deleted or removed from a document.
    • Best for showing revisions or edits in a document.
    • Pair with <ins> for clear indication of changes.
  5. <ins>
    • Use to mark text that has been inserted or added to a document.
    • Best for showing revisions or edits in a document.
    • Pair with <del> for clear indication of changes.
  6. <sup>
    • Use to format text as superscript, i.e., raised above the baseline.
    • Best for footnotes, mathematical exponents, and copyright symbols.
    • Avoid using excessively for readability and accessibility.
  7. <sub>
    • Use to format text as subscript, i.e., lowered below the baseline.
    • Best for chemical formulas, mathematical notations, and some abbreviations.
    • Avoid using excessively for readability and accessibility.
