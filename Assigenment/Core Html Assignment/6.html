02 Week - Core Html Assignment

  1. <html>
    1. The <html> tag is the root element of an HTML document.
    2. It encapsulates the entire HTML document and contains all other HTML elements.
  2. <head>
    1. The <head> tag is used to contain meta-information about the document.
    2. It includes elements like <title> , <meta>, and <link>, which provide information about the document's title, character encoding, and linked resources.
  3. <title>
    1. The <title> tag defines the title of the HTML document.
    2. The title is displayed in the browser's title bar or tab, and it is often used by search engines for indexing.
  4. <meta>
    1. The <meta> tag provides metadata about the HTML document.
    2. It can specify the character encoding, author, and other information used by browsers and search engines.
  5. <link>
    1. The <link> tag is used to link external resources like stylesheets and icon files to the HTML document.
    2. It's commonly used to include CSS stylesheets via the 'rel' attribute.
  6. <body>
    1. The <body> tag contains the visible content of the web page.
    2. It includes elements like headings, paragraphs, images, and more that are displayed to users.
  7. <h1>,<h2>,<h3>,<h4>,<h5>,<h6>
    1. Heading tags are used for creating headings of different levels, with <h1> being the highest and <h6> the lowest.
    2. They help structure content and define the importance of different sections within the document.
  8. <p>
    1. The <p> tag defines a paragraph of text.
    2. It is used to separate and format text into distinct blocks.
  9. <a>
    1. The <a> (anchor) tag creates hyperlinks.
    2. It is used to link to other web pages or resources, both internal and external.
  10. <img>
    1. The <img> tag is used to embed images in the document.
    2. It requires the 'src' attribute to specify the image source file.
  11. <ul> and <ol>
    1. <ul> and <ol> are used to create unordered and ordered lists, respectively.
    2. <ul> creates bulleted lists, while <ol> creates numbered lists
  12. <li>
    1. The <li> tag is used within <ul> and <ol> to define list items.
    2. Each <li> element represents an item within the list.
  13. <div>
    1. The <div> tag is a generic container used for grouping and styling elements.
    2. It's often used with CSS for layout and design purposes.
  14. <span>
    1. The <span> tag is another generic container, but it is an inline element.
    2. It's typically used for applying styles or scripts to a specific portion of text.
  15. <table>
    1. The <table> tag is used to create tables within the document.
    2. It includes elements like <tr>, <th>, and <td> for defining rows, headers, and data cells.
